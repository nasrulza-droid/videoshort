Login Admin dan Database Pendaftar

Halaman ini dirancang untuk GitHub Pages dengan backend eksternal Supabase. Admin login memakai Supabase Auth, dan data pendaftar disimpan di tabel database sehingga tidak lagi bergantung pada file lokal browser.

Konfigurasi Supabase belum diisi. Buka file supabase-config.js, masukkan URL dan anon key project Supabase, lalu refresh halaman ini.

Login Admin

Gunakan akun admin yang sudah dibuat di Supabase Authentication.

Model ini cocok untuk GitHub Pages karena frontend tetap statis. Hak akses admin dikontrol lewat Supabase Auth, bukan link publik semata.

Setup Singkat

  1. Buat project Supabase dan aktifkan email/password login.
  2. Jalankan SQL di file SUPABASE_SETUP.md untuk membuat tabel registrations.
  3. Edit supabase-config.js dan isi URL + anon key.
  4. Buat user admin di Supabase Authentication.
  5. Login di halaman ini, lalu impor CSV hasil Google Form ke database.
