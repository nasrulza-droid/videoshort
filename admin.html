Dashboard Sederhana Data Pendaftar

Halaman ini menampilkan data pendaftar dari file CSV hasil ekspor Google Form. Alur cepat: buka Google Form responses, download CSV, lalu upload di sini.

Catatan: karena pendaftaran utama memakai Google Form, halaman ini tidak menarik data otomatis tanpa integrasi backend/API.

Impor Data Pendaftar

Upload file CSV hasil ekspor respons Google Form.

Cara Ekspor CSV dari Google Form

  1. Buka Google Form lomba, lalu masuk ke tab Responses.
  2. Klik ikon spreadsheet untuk membuka respons di Google Sheets.
  3. Di Google Sheets, klik File > Download > Comma Separated Values (.csv).
  4. Upload file CSV tersebut ke halaman admin ini.
